# PWA Icons

วางไฟล์ไอคอนของแอปในโฟลเดอร์นี้ ตามชื่อด้านล่าง (อ้างอิงจาก `app/manifest.ts`):

| ไฟล์                       | ขนาด     | purpose  | ใช้ทำอะไร                         |
| -------------------------- | -------- | -------- | -------------------------------- |
| `icon-192.png`             | 192×192  | any      | ไอคอนทั่วไป / favicon / apple    |
| `icon-512.png`             | 512×512  | any      | ไอคอนความละเอียดสูง / splash     |
| `icon-maskable-192.png`    | 192×192  | maskable | ไอคอนแบบ adaptive (Android)      |
| `icon-maskable-512.png`    | 512×512  | maskable | ไอคอนแบบ adaptive (Android)      |

## หมายเหตุ
- ไฟล์ต้องเป็น **PNG** พื้นหลังควรทึบ (สำหรับ `any`)
- ไอคอน **maskable** ต้องเผื่อ safe zone ขอบประมาณ 10% (เนื้อหาหลักอยู่กลางวงกลม)
  เครื่องมือช่วยทำ: https://maskable.app/
- เปลี่ยนชื่อ/ขนาดได้ แต่ต้องแก้ให้ตรงกับ `app/manifest.ts` ด้วย
